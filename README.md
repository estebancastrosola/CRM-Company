# CRM-Company
CRM for Companys


## Get started
### 1. Install Ionic
`npm install -g ionic`

### 2. Clone this repo
`git clone https://github.com/estebancastrosola/CRM-Company.git`

### 3. Install modules
`cd myApp`  
`npm install`

### 4. Run mock api
`json-server --watch db.json`

### 5. Run the app
`ionic serve`
